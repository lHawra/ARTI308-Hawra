# Data Quality Assessment and Preprocessing

## Overview
This project applies data quality assessment and preprocessing techniques on the Walmart Sales dataset to prepare it for analysis and machine learning.

---

## Tasks
- Data quality checks (missing values, duplicates, data types)  
- Missing value handling strategy  
- Outlier detection using IQR  
- Normalization using Min-Max and Z-score  
- PCA for dimensionality reduction  

---

## Key Findings
- No missing or duplicate values were found  
- Outliers were detected and handled  
- Normalization improved data consistency  
- Weak correlations limit the effectiveness of PCA  

---

## Tools
Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn, Google Colab  

---

## Conclusion
The dataset was successfully cleaned and prepared. PCA was applied, but its impact is limited due to weak feature correlations.
